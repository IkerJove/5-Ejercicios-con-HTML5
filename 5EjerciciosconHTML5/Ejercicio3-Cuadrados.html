Diferencias con otros formatos gráficos

SVG e un formato gráfico baseado en vectores que representa as imaxes como valores matemáticos JPG é u raster ou formato de imaxe bitmap PNG usa código binario para comprimir os datos da imaxe Os ficheiros SVG son os preferidos para deseño gráfico porque calquera pode facilmente ver as dimensións e valores de cada elemento da imaxe. Outra vantaxe importante do formato SVG e a súa integración coas CSS Media Queries. Os layouts desenvoltos como parte dun deseño responsivo adáptanse automáticamente o tamaño da pantalla (viewport) via CSS Media Queries. Os gráficos SVG poden ser integrados nos elementos HTMLe controlados via media queries para que se poidan adaptar ó ancho da fiestra do navegaor.
